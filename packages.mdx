Iosevka is released as a large family of prebuilt packages, each targeting a specific visual style, spacing behavior, and download format. Whether you want a single bundled file for desktop use or individual TTF files for embedding in an application, there is a package format to match. All packages are available on the [GitHub Releases page](https://github.com/be5invis/Iosevka/releases).

## Packaging formats

Every Iosevka package family is available in four distribution formats:

| Format | Contents | Best for |
| --- | --- | --- |
| **Super TTC** | 1 `.ttc` file — all fonts bundled together | Desktop installation on modern operating systems |
| **TTC** | 9 `.ttc` files — one per weight | Desktop use when a single bundled file is too large |
| **TTF** | 54 `.ttf` files — one per weight, width, slope, and spacing variant | Embedding in applications; also available in an unhinted variant that reduces file size |
| **WebFont** | 1 `.css` file + 54 `.woff2` files + 54 `.ttf` files | Using Iosevka on websites |

<Tip>
  For desktop use, download the **Super TTC** package. It bundles all fonts into a single file and is the simplest way to get the complete family installed. Requires a modern operating system with TTC support.
</Tip>

## Spacing variants

Every monospace package family includes three spacing variants:

**Default** — The standard variant with ligatures enabled. Arrows, geometric symbols, and other wide glyphs occupy 2 columns, which gives them visual weight matching their semantic scale in most editors.

**Term** — A narrower variant for terminal use. Arrows and geometric symbols render as single-column characters, matching the typical convention of terminal emulators where every character is expected to be exactly one cell wide.

**Fixed** — A strictly monospace variant with no ligatures and no wide glyphs. Use this when your environment does not correctly interpret Iosevka as monospace, or when ligatures cause rendering issues.

<Note>
  Style presets apply a fixed set of character variant options at the package level. If you want to mix and match individual character shapes, use the [online customizer](https://be5invis.github.io/Iosevka/customizer) or build Iosevka from source to create your own variant configuration.
</Note>
