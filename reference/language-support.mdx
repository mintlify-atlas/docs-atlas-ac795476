Every Iosevka family — monospace and quasi-proportional — ships with the same character coverage. You do not need to choose a family based on script support; the character set is consistent across all prebuilt variants.

## Character ranges

Iosevka covers the following Unicode blocks and script ranges:

<CardGroup cols={2}>
  <Card title="Latin" icon="a">
    Full Basic Latin, Latin Extended-A, Latin Extended-B, Latin Extended Additional, and a broad selection of Latin supplement characters. Covers all accented and diacritical forms needed for European languages.
  </Card>
  <Card title="Greek and Coptic / Greek Extended" icon="circle">
    Includes **Polytonic Greek** — the full range of ancient and scholarly Greek with all breathing marks, accent combinations, and iota subscripts needed for classical texts.
  </Card>
  <Card title="Cyrillic" icon="list">
    A broad selection of Cyrillic characters covering Russian, Ukrainian, Bulgarian, Serbian, Macedonian, Belarusian, and numerous other Cyrillic-script languages.
  </Card>
  <Card title="IPA extensions" icon="mic">
    International Phonetic Alphabet symbols and phonetic extensions, including superscript letters and modifier letters used in linguistic transcription.
  </Card>
  <Card title="Punctuation and symbols" icon="type">
    General punctuation, currency symbols, letterlike symbols, arrows, mathematical operators, box-drawing characters, block elements, geometric shapes, and miscellaneous technical symbols.
  </Card>
  <Card title="Spacing modifier letters" icon="space">
    Combining diacritical marks, spacing modifier letters, and superscript/subscript forms for phonetic, mathematical, and typographic use.
  </Card>
</CardGroup>

<Tip>
  You can inspect the full glyph inventory interactively at the [Iosevka specimen page](https://be5invis.github.io/Iosevka/specimen).
</Tip>

## CJK users

Iosevka does **not** include CJK (Chinese, Japanese, Korean) characters. For CJK support, use [Sarasa Gothic](https://github.com/be5invis/Sarasa-Gothic), a companion project by the same author that combines Iosevka with the Source Han Sans CJK glyphs. Sarasa Gothic maintains the same character style and proportions as Iosevka for ASCII content, ensuring visual consistency in mixed-script environments.

<Note>
  Sarasa Gothic is maintained as a separate project and has its own release schedule, which may lag behind the latest Iosevka release.
</Note>
